## 🎮 不动点理论实现

### Banach不动点定理
```
压缩映射: T: X → X
压缩常数: 0 < k < 1
不动点: x* = T(x*)
收敛性: ||x_n - x*|| ≤ k^n ||x_0 - x*||
```

### Brouwer不动点定理
```
连续映射: f: B^n → B^n
不动点: x* = f(x*)
存在性: 紧凸集上的连续映射必有不动点
```

### Kakutani不动点定理
```
集值映射: F: X → 2^X
不动点: x* ∈ F(x*)
条件: 上半连续、非空、紧凸值
```

### Schauder不动点定理
```
紧算子: T: K → K
不动点: x* = T(x*)
条件: 紧凸集上的紧算子
```
